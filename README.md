# Walker Track

Personal web app for FDC-style ammunition and asset tracking (BOCs, POCs, launchers, pods, RSVs, tasks, fire missions, logs). Data lives in the browser using **SQLite (sql.js)** with **IndexedDB** persistence. Optional peer snapshot sync and a small serverless ingest (`api/fdc.mjs`) exist for hosted setups.

Runs anywhere you have a modern browser (desktop, tablet, Pi kiosk, etc.).
